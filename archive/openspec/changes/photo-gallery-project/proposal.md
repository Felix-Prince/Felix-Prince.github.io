## Why

需要一个简约、符合摄影审美的个人摄影作品展示网站，支持灵活的图片分类与标签管理，同时提供可配置的主题系统，方便其他摄影爱好者快速搭建类似网站。

## What Changes

- 新增摄影作品集展示页面，采用简约设计风格
- 新增图片标签数据结构（支持多标签分类，通过编辑 JSON 配置实现）
- 新增按分类/标签筛选与搜索图片功能（纯前端实现）
- 新增可配置主题系统（主题色、图标、logo、文案等）
- 新增图片时间线视图（按拍摄时间排序）
- 新增图片灯箱查看功能

## Capabilities

### New Capabilities

- `photo-gallery`: 摄影作品展示核心功能，包含网格布局、时间线视图、图片详情查看
- `photo-tagging`: 图片标签系统，通过 JSON 配置文件管理图片标签、支持多标签分类、标签筛选
- `theme-config`: 主题配置系统，支持主题色、图标、logo、站点信息等可配置项
- `photo-search`: 图片搜索与筛选功能，支持按标签、时间范围等条件搜索

### Modified Capabilities

（无现有功能需要修改）

## Impact

- 新增 `gallery/` 目录存放摄影集相关页面
- 新增配置文件 `theme.config.json` 用于主题配置
- 新增数据文件 `photos.json` 用于存储图片元数据与标签（手动编辑）
- 复用现有项目的 CSS/JS 架构与设计风格
- 图片上传与打标通过本地文件管理 + JSON 配置实现，暂不实现后台管理界面
