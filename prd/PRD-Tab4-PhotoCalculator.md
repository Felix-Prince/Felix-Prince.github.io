# PRD — Tab 4：拍摄计算器

> 优先级：P1
> 最后更新：2026-05-03

---

## 一、功能概述

纯计算工具集，无需上传照片。包含三个子计算器：黄金/蓝调时刻、ND 滤镜长曝时间、景深/超焦距。帮助摄影师在拍摄前快速做出参数决策。

## 三、功能规格

### 3.1 黄金时刻计算器

| 功能 | 规格 | 实现 |
|------|------|------|
| 自动定位 | Geolocation API 获取位置 | `navigator.geolocation.getCurrentPosition()` |
| 手动定位 | 搜索城市名或输入经纬度 | 内置城市坐标表 + 手动输入 |
| 日期选择 | 选择日期查看当日太阳轨迹 | Date Picker |
| 日出/日落 | 计算当日日出日落时间 | `suncalc` |
| 黄金时刻 | 日出后/日落前约 30~60min | `suncalc` + 自定义阈值 |
| 蓝调时刻 | 民用黎明/民用黄昏 | `suncalc` (civil twilight) |
| 月出/月落 | 当日月亮升降时间 | `suncalc` moonCalc |
| 月相 | 当前月相和光照百分比 | `suncalc` |
| 太阳轨迹图 | 一天中太阳方位角 + 仰角曲线 | `suncalc` + Canvas 绘制 |

**输出示例**：
```
📍 上海 | 2026-05-03
────────────────────
🌅 日出    05:18
🟡 黄金时刻 05:18 - 06:12
🔵 蓝调时刻 04:42 - 05:18
🌇 日落    18:45
🟡 黄金时刻 17:52 - 18:45
🔵 蓝调时刻 18:45 - 19:19
🌙 月出    20:30 | 月相: 上弦月 45%
```

### 3.2 ND 滤镜计算器

**输入**：

| 参数 | 说明 | 控件 |
|------|------|------|
| 基础快门速度 | 无 ND 滤镜时的正常曝光时间 | Select（1/8000s ~ 30s）或自定义 |
| ND 档位 | 见下表 | Select 或点击按钮 |

**ND 档位预设**：

| 名称 | 档位 | 倍率 |
|------|------|------|
| ND2 | 1 | 2× |
| ND4 | 2 | 4× |
| ND8 | 3 | 8× |
| ND16 | 4 | 16× |
| ND32 | 5 | 32× |
| ND64 | 6 | 64× |
| ND400 | 9 | 400× |
| ND1000 | 10 | 1000× |
| 自定义 | 用户输入 | 2^n |

**输出**：

| 参数 | 公式 | 示例 |
|------|------|------|
| 长曝时间 | `基础时间 × 2^档位` | 1/60s + 10档 = 16s |
| 可读格式 | 自动转「X 分 Y 秒」 | 16s → "16 秒"，64s → "1 分 4 秒" |

### 3.3 景深 / 超焦距计算器

**输入**：

| 参数 | 范围 | 控件 |
|------|------|------|
| 焦距 | 14mm ~ 600mm | 滑块 + Input |
| 光圈 | f/1.4 ~ f/22 | 滑块 + Select |
| 对焦距离 | 0.5m ~ ∞ | Input + ∞ 按钮 |
| 传感器尺寸 | 全画幅 / APS-C(Canon) / APS-C(其他) / M43 / 中画幅 | Select |

**传感器 + 弥散圆数据**（`sensor-sizes.ts`）：

| 传感器 | 弥散圆 (mm) | 裁切系数 |
|--------|------------|---------|
| 全画幅 | 0.030 | 1.0× |
| APS-C (Canon) | 0.019 | 1.6× |
| APS-C (其他) | 0.020 | 1.5× |
| M43 | 0.015 | 2.0× |
| 中画幅 | 0.045 | 0.64× |

**输出**：

| 参数 | 公式 | 说明 |
|------|------|------|
| 近界 | `Dn = s × (H - f) / (H + s - 2f)` | 最近清晰距离 |
| 远界 | `Df = s × (H - f) / (H - s)` | 最远清晰距离（H < s 时为 ∞） |
| 总景深 | `Df - Dn` | 景深范围 |
| 超焦距 | `H = f² / (N × c) + f` | 超焦距距离 |

**可视化图**：用 Canvas 绘制焦点 + 景深范围示意图：

```
    近界          焦点          远界
      |←───────────|───────────→|
      |     清晰区域             |
──|───|───────────|───────────|──→
  0m  2.1m        5m          ∞
                   H = 4.2m
```

## 七、风险

| 风险 | 缓解 |
|------|------|
| Geolocation 权限被拒 | 提供手动输入城市/经纬度 |
| SunCalc 精度 | 民用级精度足够，误差 ±1min |
| 城市坐标表不完整 | 优先覆盖一二线城市，支持手动输入经纬度 |
| 景深公式为近似值 | 标注"近似值"，实际受镜头光学特性影响 |
