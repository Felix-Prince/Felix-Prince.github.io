# PRD — Tab 4：拍摄计算器

> 优先级：P1
> **目标用户：富士 APS-C 画幅用户优先**
> 最后更新：2026-05-14

---

## 一、功能概述

纯计算工具集，无需上传照片。包含三个子计算器：黄金/蓝调时刻、ND 滤镜长曝时间、景深/超焦距。**APS-C（富士）为默认传感器**，所有计算器默认展示等效 35mm 焦距。帮助摄影师在拍摄前快速做出参数决策。

## 二、布局

## 三、功能规格

### 3.1 黄金时刻计算器

| 功能 | 规格 | 实现 |
|------|------|------|
| 自动定位 | Geolocation API 获取位置 | `navigator.geolocation.getCurrentPosition()` |
| 手动定位 | 搜索城市名或输入经纬度 | 内置城市坐标表 + 前端模糊匹配 |
| 日期选择 | 选择日期查看当日太阳轨迹 | Date Picker |
| 日出/日落 | 计算当日日出日落时间 | `suncalc` |
| 黄金时刻 | 日出后/日落前约 30~60min | `suncalc` + 自定义阈值 |
| 蓝调时刻 | 民用黎明/民用黄昏 | `suncalc` (civil twilight) |
| 月出/月落 | 当日月亮升降时间 | `suncalc` moonCalc |
| 月相 | 当前月相和光照百分比 | `suncalc` |

**输出示例**：
```
📍 上海 | 2026-05-03
────────────────────
🌅 日出    05:18
🟡 黄金时刻 05:18 - 06:12
🔵 蓝调时刻 04:42 - 05:18
🌇 日落    18:45
🟡 黄金时刻 17:52 - 18:45
🔵 蓝调时刻 18:45 - 19:19
🌙 月出    20:30 | 月相: 上弦月 45%
```

### 3.2 ND 滤镜计算器

**输入**：

| 参数 | 说明 | 控件 |
|------|------|------|
| 基础快门速度 | 无 ND 滤镜时的正常曝光时间 | Select（1/8000s ~ 30s）或自定义 |
| ND 档位 | 见下表 | Select 或点击按钮 |

**ND 档位预设**：

| 名称 | 档位 | 倍率 |
|------|------|------|
| ND2 | 1 | 2× |
| ND4 | 2 | 4× |
| ND8 | 3 | 8× |
| ND16 | 4 | 16× |
| ND32 | 5 | 32× |
| ND64 | 6 | 64× |
| ND400 | 9 | 400× |
| ND1000 | 10 | 1000× |
| 自定义 | 用户输入 | 2^n |

**输出**：

| 参数 | 公式 | 示例 |
|------|------|------|
| 长曝时间 | `基础时间 × 2^档位` | 1/60s + 10档 = 16s |
| 可读格式 | 自动转「X 分 Y 秒」 | 16s → "16 秒"，64s → "1 分 4 秒" |

**电子 ND 提示**（富士相机专属）：当计算结果 ≤ 1s 时，提示"富士部分机型支持电子 ND 快门，无需物理 ND 滤镜即可实现此效果"。

### 3.3 景深 / 超焦距计算器

**输入**：

| 参数 | 范围 | 控件 | 说明 |
|------|------|------|------|
| 焦距 | 14mm ~ 600mm | 滑块 + Input | 自动显示 35mm 等效值（如 23mm → 35mm） |
| 光圈 | f/1.4 ~ f/22 | 滑块 + Select | |
| 对焦距离 | 0.5m ~ ∞ | Input + ∞ 按钮 | |
| 传感器尺寸 | APS-C 富士(默认) / 全画幅 / APS-C(Canon) / M43 / 中画幅 | Select | 默认选择 APS-C (1.5×) |

**传感器 + 弥散圆数据**（`sensor-sizes.ts`）：

| 传感器 | 弥散圆 (mm) | 裁切系数 | 默认选中 |
|--------|------------|---------|---------|
| 全画幅 | 0.030 | 1.0× | |
| APS-C (Fujifilm 等) | 0.020 | 1.5× | ✅ 默认 |
| APS-C (Canon) | 0.019 | 1.6× | |
| M43 | 0.015 | 2.0× | |
| 中画幅 | 0.045 | 0.64× | |

**输出**：

| 参数 | 公式 | 说明 |
|------|------|------|
| 近界 | `Dn = s × (H - f) / (H + s - 2f)` | 最近清晰距离 |
| 远界 | `Df = s × (H - f) / (H - s)` | 最远清晰距离（H < s 时为 ∞） |
| 总景深 | `Df - Dn` | 景深范围 |
| 超焦距 | `H = f² / (N × c) + f` | 超焦距距离 |
| 等效焦距 | `f × 裁切系数` | 自动显示 35mm 全画幅等效焦距 |

**可视化图**：用 Canvas 绘制焦点 + 景深范围示意图：

```
    近界          焦点          远界
      |←───────────|───────────→|
      |     清晰区域             |
──|───|───────────|───────────|──→
  0m  2.1m        5m          ∞
                   H = 4.2m
```

### 3.4 焦段推荐 + 拍摄指南

基于用户输入的焦距，自动推荐适合的拍摄题材、建议拍摄距离，并列出对应的富士/副厂镜头参考。

**推荐逻辑**（`focal-length-guide.ts`）：

| 焦段范围 (APS-C) | 等效 35mm | 适合题材 | 建议距离 | 富士原生镜头举例 |
|-----------------|-----------|---------|---------|----------------|
| 8~14mm | 12~21mm | 星空/风光/建筑 | 0.5~2m | XF 8mm f/3.5, XF 10-24mm f/4 |
| 16~23mm | 24~35mm | 街拍/人文/环境人像 | 1~3m | XF 16mm f/1.4, XF 18mm f/2, XF 23mm f/1.4 |
| 27~35mm | 40~53mm | 人文/半身人像 | 1.5~4m | XF 27mm f/2.8, XF 33mm f/1.4, XF 35mm f/1.4 |
| 50~56mm | 75~85mm | 人像/特写 | 2~5m | XF 50mm f/2, XF 56mm f/1.2 |
| 70~90mm | 105~135mm | 人像/静物/压缩 | 3~8m | XF 80mm f/2.8, XF 90mm f/2 |
| 100~200mm | 150~300mm | 远摄/体育/生态 | 5~30m+ | XF 100-400mm f/4.5-5.6, XF 150-600mm f/5.6-8 |

> **变焦镜头**：变焦镜头取两端焦段分别推荐，中间焦段覆盖范围自动推断。
> **副厂镜头**：支持 Tamron、Sigma 等副厂镜头，焦段物理规律通用（如 Sigma 18-50mm f/2.8 等效 27-75mm）。
> **距离标注**：建议距离为参考值，实际取决于构图需求和主体大小。

**交互**：

| 功能 | 规格 | 实现 |
|------|------|------|
| 自动推荐 | 焦距滑块变化时自动更新推荐卡片 | 状态驱动 |
| 镜头列举 | 显示对应焦段的富士原生 + 副厂镜头 | 内置镜头表查询 |
| 等效焦距 | 所有焦距同时显示 APS-C 实际值和 35mm 等效值 | `f × 1.5` 自动换算 |
| 拍摄提示 | 根据焦段显示"建议距离 X~Y 米" | 距离矩阵映射 |

## 七、风险

| 风险 | 缓解 |
|------|------|
| Geolocation 权限被拒 | 提供手动输入城市/经纬度 |
| SunCalc 精度 | 民用级精度足够，误差 ±1min |
| 城市坐标表不完整 | 优先覆盖一二线城市，支持手动输入经纬度 |
| 景深公式为近似值 | 标注"近似值"，实际受镜头光学特性影响 |
| 镜头数据不完整 | 首批覆盖富士主流原生镜头，副厂后续补全 |
| 推荐距离为经验值 | 标注"参考值"，实际取决于构图需求 |
