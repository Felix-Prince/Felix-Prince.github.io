> 优先级：P1
> 最后更新：2026-06-05

---

## 一、功能概述

拍摄前的灵感参考库。汇集各类拍摄方案，每个方案包含：主题描述、推荐器材、拍摄参数、拍摄思路、参考图、小贴士。支持按分类筛选和关键词搜索，点击卡片进入完整详情页。

## 三、功能规格

### 3.1 画廊模式

| 功能 | 规格 | 实现 |
|------|------|------|
| 卡片展示 | 每个方案一张卡片，含封面/标题/分类/难度/预估时间 | CSS Grid |
| 分类筛选 | 顶部分类标签栏，点击切换 | state 驱动 |
| 关键词搜索 | 搜索框实时过滤方案标题/描述/标签 | includes 匹配 |
| 响应式 | 桌面端 3 列，平板 2 列，移动端 1 列 | CSS Grid auto-fill |

### 3.2 详情模式

| 功能 | 规格 | 实现 |
|------|------|------|
| 内容渲染 | 渲染 sections 数组（text/image/heading/tip/settings-table） | SectionRenderer |
| 器材展示 | 推荐机身/镜头/配件列表 | PlanDetail |
| 参数展示 | 推荐光圈/快门/ISO/焦距/白平衡 | PlanDetail |
| 返回画廊 | 点击返回按钮，保留筛选/搜索状态 | state 提升 |

### 3.3 数据

| 功能 | 规格 | 实现 |
|------|------|------|
| 数据存储 | TypeScript 数据文件 + 结构化 section 数组 | src/data/shooting-plans.ts |
| 图片占位 | 封面用 CSS gradient + emoji，零存储成本 | inline style |
| 外部图片 | 参考图预留 URL 字段，后续可外链 | src: string |

## 四、组件结构

```
src/components/cameravision/ShootingPlan/
├── ShootingPlan.tsx           # 主入口：状态管理 + 模式切换
├── PlanCard.tsx               # 方案卡片
├── PlanDetail.tsx             # 方案详情
├── PlanFilters.tsx            # 分类筛选 + 搜索
└── SectionRenderer.tsx        # 内容 section 渲染
```

## 五、数据结构

```ts
type PlanSection =
  | { type: 'text'; content: string }
  | { type: 'image'; src: string; caption?: string; alt: string }
  | { type: 'heading'; content: string; level?: 2 | 3 }
  | { type: 'tip'; content: string }
  | { type: 'settings-table'; rows: { label: string; value: string }[] };
```

完整类型定义见 `src/data/shooting-plans.ts`。

## 八、风险

| 风险 | 缓解 |
|------|------|
| 方案数据维护成本 | 结构化数据 + 代码即数据，编辑方便 |
| 图片外链失效 | 预留字段，URL 统一管理，可批量替换 |
| 内容质量参差 | 初期由用户自身把关，方案精选 |
