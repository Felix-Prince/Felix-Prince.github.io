# PRD — Tab 4：摆姿参考库

> 优先级：P1
> 最后更新：2026-06-05

---

## 一、功能概述

人像拍摄时的摆姿灵感库。提供 5 大类（站/坐/蹲/躺/趴）共 25 个姿势的 SVG 人体轮廓图，每个姿势配有拍摄描述和提示。支持线稿/剪影双风格切换，与 Tab 3 拍摄方案联动推荐姿势。

---

## 二、视觉风格

**Tube 轮廓线稿**：基于关节坐标算法生成的人体轮廓，清晰展示肢体摆放关系。

- 无五官、无头发、无服装细节，仅呈现身体轮廓与姿态
- **躯干**：肩→腰→胯的沙漏形曲线（腰部明显收窄）
- **四肢**：上宽下窄的 Tube 管状轮廓（肩/胯处粗 → 手/脚处细）
- **头部**：简化椭圆，女性化头身比
- **双渲染模式**：
  - 线稿：黑色描边，空心轮廓，干净清晰
  - 剪影：黑色填充，关节处覆盖圆消除缝隙
- 零图片存储，数据为关节坐标 + 算法实时渲染

## 三、功能规格

### 3.1 姿势数据

5 大基础姿态 × 5 变体 = 25 个姿势：

| ID | 分类 | 5 个变体 |
|----|------|---------|
| standing | 站姿 | 自然站立、侧身回眸、倚墙而立、手扶帽檐、交叉手臂 |
| sitting | 坐姿 | 椅坐、盘腿坐、侧坐、高脚凳坐、坐姿托腮 |
| squatting | 蹲姿 | 全蹲、单膝跪地、半蹲、蹲身手触地、蹲姿侧身 |
| lying | 躺姿 | 仰躺、侧躺、屈膝侧躺、抬腿躺、仰躺举手 |
| prone | 趴姿 | 平趴、肘撑趴、侧趴、趴姿抬腿、趴姿撑头 |

每个姿势包含：
- `figure`: 关节坐标（头/颈/肩×2/肘×2/手×2/胯×2/膝×2/脚×2）
- `viewBox`: SVG 视口
- `scenes`: 适用场景标签（portrait/street/nature/indoor）
- `description`: 姿势说明
- `tip`: 拍摄引导提示
- `difficulty`: 难度（easy/medium/hard）

### 3.2 渲染引擎（PoseSvg）

**Tube 算法**：
给定两个关节坐标 `from` / `to` 和两端宽度 `w1` / `w2`，计算垂直于肢体方向的偏移向量，生成四边形 SVG path。

```
from(宽 w1) ────────────────── to(宽 w2)
  ↑ 垂直偏移 ±w/2 生成四边形轮廓
```

**各部位宽度比例**（viewBox 坐标系）：

| 部位 | 近躯干端 | 远躯干端 |
|------|---------|---------|
| 颈部 | 3 | 4.5 |
| 上臂 | 6.5 (肩) | 4.5 (肘) |
| 前臂 | 4.5 (肘) | 2.8 (手) |
| 大腿 | 8.5 (胯) | 5.5 (膝) |
| 小腿 | 5.5 (膝) | 3.5 (脚) |

**躯干算法**：
1. 取肩线中点和胯线中点，得到躯干中心线
2. 计算垂直于中心线的方向向量（指向左侧）
3. 肩/胯处宽度 = 骨架距离 + 扩展量（5px / 6px）
4. 腰部控制点向内收 2.5px，形成 Q 二次贝塞尔曲线收腰

### 3.3 画廊模式

| 功能 | 规格 |
|------|------|
| 卡片展示 | SVG 渲染图 + 名称 + 分类 + 难度星级 + 场景标签 |
| 分类筛选 | 顶部标签栏，按 5 大基础姿态筛选 |
| 风格切换 | 线稿/剪影一键切换，偏好存入 localStorage |
| 场景标签 | 人像/街拍/户外/室内 pill 标签 |

### 3.4 详情模式

| 功能 | 规格 |
|------|------|
| 大 SVG 渲染 | viewBox 自适应，可切换风格 |
| 拍摄描述 | 姿势说明 + 适用场景标签 |
| 小贴士 | 拍摄时的引导提示（高亮边框） |
| 方案联动 | 底部展示关联的拍摄方案（Phase 2） |

### 3.5 Tab 3 联动

方案详情页底部展示 2-3 个推荐姿势 SVG 缩略图，点击切换至 Tab 4 并打开对应姿势详情。
