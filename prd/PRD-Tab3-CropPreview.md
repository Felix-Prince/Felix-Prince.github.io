# PRD — Tab 3：裁切预览

> 优先级：P1
> 最后更新：2026-05-03

---

## 一、功能概述

上传照片，选择目标社交平台，实时预览裁切框和安全区叠加，拖拽调整裁切位置，按目标比例裁切导出。重点适配中国社交平台（小红书、微信朋友圈）。

## 四、功能规格

### 4.1 裁切框叠加

| 功能 | 规格 | 实现 |
|------|------|------|
| 裁切框渲染 | 在照片上叠加平台对应比例的裁切框 | Canvas / CSS clip-path |
| 遮罩效果 | 裁切外区域半透明暗色遮罩 | 叠加层 opacity |
| 实时切换 | 切换平台立即更新裁切框 | 状态驱动重绘 |
| 比例标注 | 裁切框角落显示当前比例和目标像素 | Canvas 文字绘制 |

### 4.2 安全区叠加

| 功能 | 规格 | 实现 |
|------|------|------|
| 安全区开关 | 可切换显示/隐藏安全区 | Checkbox |
| 头像遮挡 | 平台个人头像位置半透明红色区域 | 预设 overlay 模板 |
| UI 遮挡 | 平台按钮/文字等 UI 遮挡区域 | 预设 overlay 模板 |
| 分平台模板 | 每个平台独立的安全区数据 | `crop-presets.ts` 数据驱动 |

**安全区数据结构**：
```ts
interface SafeZone {
  platform: string;
  ratio: string;
  overlays: Array<{
    x: number;      // 百分比位置
    y: number;
    width: number;  // 百分比尺寸
    height: number;
    label: string;  // "头像" / "点赞按钮" 等
  }>;
}
```

### 4.3 拖拽调整

| 功能 | 规格 | 实现 |
|------|------|------|
| 裁切框拖拽 | 上下/左右拖动调整裁切位置 | 鼠标/触摸拖拽 |
| 边界限制 | 裁切框不能超出照片边界 | 坐标 clamp |
| 实时更新 | 拖拽过程中遮罩实时更新 | requestAnimationFrame |

### 4.4 裁切导出

| 功能 | 规格 | 实现 |
|------|------|------|
| 裁切导出 | 按目标比例裁切并导出对应尺寸 | Canvas drawImage + crop |
| 格式 | JPG | `canvas.toBlob('image/jpeg')` |
| 多平台 | 可分别导出多个平台裁切版本 | 循环处理 |
| 文件名 | `cameravision-{platform}-{ratio}-{timestamp}.jpg` | — |

## 八、风险

| 风险 | 缓解 |
|------|------|
| 安全区数据不准 | 以主流平台实际截图为基准，社区反馈修正 |
| 平台尺寸变更 | 数据驱动，更新 `crop-presets.ts` 即可 |
| 非整数比例裁切精度 | 使用 Canvas drawImage 的 sx/sy/sw/sh 参数精确裁切 |
