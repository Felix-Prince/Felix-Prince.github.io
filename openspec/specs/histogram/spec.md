## ADDED Requirements

### Requirement: RGB 三通道直方图

系统 SHALL 对上传照片生成红、绿、蓝独立通道的 256 级直方图。

#### Scenario: 标准照片直方图统计

- **WHEN** 用户上传一张标准尺寸照片
- **THEN** 系统使用 Canvas `getImageData` 逐像素统计 R/G/B 通道在 0-255 区间的分布
- **THEN** 系统渲染三条独立曲线（红色/绿色/蓝色），横轴为亮度级别（0-255），纵轴为像素数量（归一化）

#### Scenario: 4K 大图降采样

- **WHEN** 照片宽度超过 2000px
- **THEN** 系统先将照片缩放至 2000px 宽度再统计直方图
- **THEN** 缩略统计的计算耗时不超过 500ms

#### Scenario: 直方图渲染

- **WHEN** 直方图数据计算完成
- **THEN** 系统使用 Canvas 2D API 绘制直方图曲线
- **THEN** 三通道曲线叠加在同一坐标系中
- **THEN** 图表包含坐标轴标签和网格辅助线

### Requirement: 亮度直方图

系统 SHALL 对上传照片生成整体亮度分布直方图。

#### Scenario: 亮度计算

- **WHEN** 照片像素数据已读取
- **THEN** 系统使用加权公式 `Y = R×0.299 + G×0.587 + B×0.114` 计算每个像素的亮度值
- **THEN** 系统统计亮度值在 0-255 区间的分布
- **THEN** 亮度直方图与 RGB 直方图在同一区域展示（独立面板或分 Tab）

### Requirement: 过曝警告

系统 SHALL 对照片中亮度值 ≥ 250 的高光溢出区域进行视觉标注。

#### Scenario: 检测并标注过曝区域

- **WHEN** 照片直方图计算完成
- **THEN** 系统遍历像素，标记所有 R/G/B 三通道均 ≥ 250 的像素
- **THEN** 系统在照片缩略图预览上叠加红色半透明闪烁遮罩层
- **THEN** 系统显示过曝像素占比百分比

#### Scenario: 无过曝区域

- **WHEN** 照片中无像素达到过曝阈值
- **THEN** 系统显示"无过曝区域"提示
- **THEN** 系统不叠加警告遮罩

### Requirement: 欠曝警告

系统 SHALL 对照片中亮度值 ≤ 5 的阴影溢出区域进行视觉标注。

#### Scenario: 检测并标注欠曝区域

- **WHEN** 照片直方图计算完成
- **THEN** 系统遍历像素，标记所有 R/G/B 三通道均 ≤ 5 的像素
- **THEN** 系统在照片缩略图预览上叠加蓝色半透明闪烁遮罩层
- **THEN** 系统显示欠曝像素占比百分比

#### Scenario: 无欠曝区域

- **WHEN** 照片中无像素达到欠曝阈值
- **THEN** 系统显示"无欠曝区域"提示
- **THEN** 系统不叠加警告遮罩
